This repository implements a high-fidelity diagnostic framework for the identification of clandestine water connections in pressurized networks through the synergic application of **Digital Signal Processing (DSP)** and **Supervised Machine Learning**. The architecture is bifurcated into an **Advanced Training Suite**, which features a robust **Data Augmentation** pipeline designed to mitigate data scarcity through synthetic signal generation (via Gaussian noise injection and temporal warping), and a specialized **Inference Engine** for real-time bypass detection. Leveraging a comprehensive feature engineering protocol, the system extracts critical multidimensional descriptors, including **Discrete Wavelet Transform (DWT)** coefficients for transient localization, **Fast Fourier Transform (FFT)** sub-band energy distributions for spectral characterization, and non-linear exponential decay constants ($\tau$) derived from hydraulic pressure transients. By deploying optimized **Support Vector Machine (SVM)** and **Random Forest (RF)** architectures, the framework provides a scalable, automated solution for Non-Revenue Water (NRW) reduction, facilitating the precise classification of pipeline integrity from raw telemetry data using serialized `joblib` models and standardized preprocessing.
